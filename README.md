# Python Basics and Office Mini Projects
This repository contains beginner Python programs created for learning and practice. 
## Projects included
1. **Even or Odd Number Checker** – Simple Python program to check if a number is even or odd.  
2. **Basic Calculator** – Program to perform addition, subtraction, multiplication, and division.  
3. **Employee Attendance & Salary Record System** – Simple project for HR/office workflow:
   - Records employee details
   - Calculates monthly salary based on attendance
   - Demonstrates data handling and accuracy
